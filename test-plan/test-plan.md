# Test Plan for Login Functionality

## System Under Test (SUT)
https://the-internet.herokuapp.com/login

## Scope
Testing the login functionality, including valid and invalid credential scenarios, form validation, and error message display.

## Test Types
- Manual functional testing
- Negative testing (invalid inputs)
- API testing (basic checks using Postman)

## Test Environment
- Browsers: Chrome (latest), Firefox (latest)
- Operating Systems: Windows 10, macOS Monterey

## Risks
- Limited user roles for testing
- Limited time to cover all edge cases

## Entry Criteria
- Application is accessible online
- Test environment is configured
- Test data is prepared

## Exit Criteria
- All critical and major bugs are resolved or documented
- All planned test cases executed
- No severe blocking issues remain
